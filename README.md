# NEXA

NEXA is a cross-platform shell environment designed to provide an easy and flexible command-line experience for developers and users.

### Windows
1. Download the latest release:
   - [NEXA_Shell_Windows.zip](https://github.com/WitheXee69/Project_NEXA/releases/latest/download/NEXA_Shell_Win64.zip) or

### Linux
1. Ensure Python 3 is installed
2. Download the latest release:
    ```bash
   wget -L https://github.com/WitheXee69/Project_NEXA/releases/latest/download/NEXA_Shell_Linux.zip
3. Extract the zip file:
    ```bash
   unzip NEXA_Shell_Linux.zip -d NEXA_Shell
4. Navigate to the extracted directory and run the shell:
    ```bash
   cd NEXA_Shell
5. Make the launcher executable and run it:
    ```bash
   chmod +x nexa_Linux_bootstrapper.sh
6. Run the launcher:
    ```bash
    ./nexa_Linux_bootstrapper.sh
   
## ðŸš€ Usage

After launching NEXA, you can use shell commands like:
- `scan` (alias for `dir` on Windows and `ls` on Linux)
- `warp` (alias for `cd` on Windows and Linux)
- `here` (alias for `cd` on Windows and `pwd` on Linux)
- `trash` (alias for `rm` on Windows and Linux)
- `forge` (alias for `touch` on Linux)
- `execute` (To execute .nexa file like Windows batch or Linux shell script)
- `reveal` (alias for `cat` on Linux and `type` on Windows)
- `say` (alias for `echo` on Linux and Windows)
- `netinfo` (alias for `ipconfig` on Windows and `ifconfig` on Linux)
- `construct` (alias for `mkdir` on Linux and Windows)
- `whoami` (To get information about NEXA and the system)
- `help` (Currently shows a list of available commands)
- And more to come!
